# 📋 Task Flow — Gestor de Tareas

**Task Flow** es una aplicación web de gestión de tareas estilo tablero Kanban, desarrollada con HTML, CSS y JavaScript puro. Permite crear tareas con prioridad y fecha límite, y moverlas entre tres estados: **Pendiente**, **En Progreso** y **Completada**.

---

## 🚀 Características

- Crear tareas con descripción, prioridad (Alta / Media / Baja) y fecha límite
- Tablero Kanban con tres columnas: **Pendientes**, **En Progreso** y **Completadas**
- Mover tareas entre columnas con botones de acción
- Eliminar tareas en cualquier momento
- Contador de tareas por columna actualizado en tiempo real
- Indicador visual de prioridad por color (rojo, amarillo, verde)
- Diseño responsivo adaptado a distintos tamaños de pantalla

## ▶️ Cómo usar

1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` directamente en tu navegador — no requiere instalación ni servidor.
3. Completa el formulario con la descripción, prioridad y fecha límite de la tarea.
4. Haz clic en **Agregar Tarea** para añadirla al tablero en la columna *Pendientes*.
5. Usa los botones de cada tarjeta para moverla entre columnas o eliminarla.
